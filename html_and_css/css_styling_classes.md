# CSS Styling Classes

## What is CSS class?

- A CSS class is used to apply styles to one or more HTML elements.

- It is written using a dot (.) in CSS.

```css
<p class="red-text">This text is red.</p>
```

## Styling classes

- Text styling

```css
.text-center {
    text-align: center;
}
```

- Background styling

```css
.bg-yellow {
    background-color: yellow;
}
```
- Spacing styling

```css
.margin-10 {
    margin: 10px;
}

.padding-20 {
    padding: 20px;
}
```

- Width and height styling

```css
.w-50 {
    width: 50%;
}

.h-100 {
    height: 100px;
}
```

## Conclusion

- CSS styling classes help us reuse the same styles for many elements, which saves time and avoids repeating code.

- They keep HTML and CSS separate, making the code cleaner and easier to maintain.

- Using classes properly makes websites consistent, organized, and easier to manage as the project.
