# How to Search on MDN (Mozilla Developer Network)

## What is MDN?

MDN stands for **Mozilla Developer Network**.  
It is a reliable website for learning web technologies like:

- HTML  
- CSS  
- JavaScript  
- Web APIs  

MDN has **clear explanations**, **simple examples**, and is used by developers around the world.

MDN link: https://developer.mozilla.org/

---

## Why Use MDN?

- Official and trusted documentation  
- Easy examples  
- Updated content  
- Clear explanation for beginners

MDN helps you understand how web languages work.

---

## How to Search in MDN

### Step 1 — Go to MDN
Open your browser and visit:
https://developer.mozilla.org/

---

### Step 2 — Use the Search Bar

At the top, there is a **search box**.

Type what you are looking for.

#### Example searches:

| What You Want | What To Type |
|---------------|-------------|
| Learn loops in JavaScript | `JavaScript loops` |
| Function return | `function return JavaScript` |
| forEach loop | `forEach MDN` |
| Objects | `JavaScript objects` |
| Arrays | `array methods MDN` |

After typing your term, press **Enter**.

---

## How to Read MDN Results

When you search:

1. The first result is usually the **official MDN page**
2. Click the result title
3. You will see:
   - Simple explanations
   - Examples
   - Interactive code blocks
   - Browser compatibility tables

---

## Summary


- MDN is one of the best places to learn web development.  
To use it:

1. Go to MDN
2. Type your keyword in the search box
3. Add “MDN” if results are not clear
4. Open the best result
5. Read simple examples and explanations

- MDN helps you learn web technologies in a clear and easy way.
